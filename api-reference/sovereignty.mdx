## `GET /api/v1/sovereignty`

Returns technology sovereignty assessments for current dependencies and a list of removed technologies. Sovereignty evaluation ensures the Mukoko ecosystem avoids vendor lock-in and prioritizes open-source, self-hostable infrastructure.

### Response Shape

| Field | Type | Description |
|---|---|---|
| `assessments` | `array` | Sovereignty evaluations for current technologies |
| `removed` | `array` | Technologies removed from the stack with rationale |

The sovereignty principle: **"Speed is rented. Truth is owned."** Edge caches (Cloudflare KV, Durable Objects) provide speed but are ephemeral and managed. Sovereign databases (Supabase, ScyllaDB, Web3 Pod) own the truth and can be self-hosted.

### Example Assessments

```json
{
  "technology": "Supabase",
  "category": "cloud",
  "verdict": "acceptable",
  "rationale": "Open-source, self-hostable. Postgres underneath. Exit strategy: migrate to raw Postgres.",
  "selfHostable": true,
  "openSource": true
}
```

```json
{
  "technology": "ScyllaDB",
  "category": "database",
  "verdict": "sovereign",
  "rationale": "AGPL 3.0 licensed. Self-hosted on Fly.io in Johannesburg. Non-relational source of truth.",
  "selfHostable": true,
  "openSource": true
}
```

```json
{
  "technology": "Cloudflare",
  "category": "edge",
  "verdict": "acceptable",
  "rationale": "Managed edge platform. Used for ephemeral caches (KV, Durable Objects), not source of truth. Speed is rented.",
  "selfHostable": false,
  "openSource": false
}
```

### Example Removed Technology

```json
{
  "technology": "Firebase",
  "reason": "Proprietary lock-in. No self-hosting. Data stored on Google infrastructure outside Africa.",
  "replacedBy": "Supabase + CouchDB"
}
```

### Example

```bash
curl https://design.nyuchi.com/api/v1/sovereignty
```
